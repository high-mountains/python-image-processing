## watershed.py

### description
Image segmented by watershed algorithm

### preparation
Download sample photo.  
Execute following command:

```sh
$ wget https://docs.opencv.org/3.1.0/water_coins.jpg
```

### usage
```sh
$ python watershed.py water_coins.jpg
```
